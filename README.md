# Weducation

API is hosted in : https://weducation.herokuapp.com/

### How to run
Clone and move to the root directory
```sh
git clone <reponame>
cd <reponame>
```
Run api/server side (you should be on the root directory)
```sh
cd server
npm install
npm start
```
Run client (you should be on the root directory)
```sh
cd client
npm install
npm start
```
